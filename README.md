# Aplicativo de controle de gastos

## Esse aplicativo foi feito com React Native + TypeScript + Firebase, feita para ajudar as peesoas a terem um controle melhor de seus gastos e lucros

## Instalação

### Instalar o NodeJs

### Entrar na pasta do projeto e digitar npm install

### Para rodar o app - npm start

### Depois só rodar o app no emulador apertando a tecla A ou pelo celular mesmo apontando a camero do celular para o qrCode
